# Solar Energy Analytics & Predictive Maintenance Dashboard

An end-to-end Machine Learning project that analyzes solar panel performance and predicts AC power output using weather sensor data.

The project includes an interactive Streamlit dashboard for monitoring solar panel efficiency and predicting power generation.
Based on efficiency, it tells whether Panel Cleaning is required or not.

---

## Features

- Solar power prediction using Random Forest regression
- Interactive Streamlit dashboard
- Weather condition input sliders
- System efficiency monitoring
- Predictive maintenance recommendation
- Data visualization and model analysis
